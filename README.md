# Hardik Patil — MERN Portfolio Website

## Overview

This is my personal portfolio website built using the MERN stack to showcase my projects, skills, experience, and professional work.
It also includes an admin dashboard that allows me to update portfolio content dynamically without changing frontend code.

The goal of this portfolio is to present my development work professionally for freelancing, internships, and career opportunities.

---

## Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* React Router
* Axios
* Lucide Icons

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Nodemailer (Contact email notifications)
* Multer (Image & Resume uploads)

---

## Features

### Portfolio Website

* Responsive modern UI
* Projects showcase with categories
* Experience & achievements section
* Blog integration
* Contact form (DB + Email notification)
* Social media links
* Dynamic footer & navbar content

### Admin Dashboard

* Secure admin access
* Update profile, projects, experience
* Upload resume & profile image

## Installation (Local Setup)

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

### 2. Install Frontend

```bash
cd client
npm install
npm run dev
```

### 3. Install Backend

```bash
cd server
npm install
npm run start
```

---

## Environment Variables (.env)

Create `.env` inside the **server** folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
EMAIL_USER=your_email
EMAIL_PASS=your_app_password
ADMIN_SECRET=your_admin_secret
```
